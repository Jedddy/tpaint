# TPaint

Mini Paint in your terminal.

## Installation

```
$ git clone github.com/Jedddy/tpaint
```

## Usage

```
$ cd tpaint
$ pip install -r requirements.txt (for windows only)
$ python main.py
```

## Keys
- Q - Quit
- C - Clear
- V - Use keyboard cursor
